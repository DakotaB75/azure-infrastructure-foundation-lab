# Code of Conduct

## Professional Environment

This repository promotes a professional, respectful and collaborative technical environment focused on Microsoft Azure infrastructure, governance and cloud operational practices.

---

## Expected Behavior

Contributors and participants are expected to:

- Maintain respectful communication
- Encourage constructive technical discussion
- Support collaborative learning
- Use professional language
- Respect operational and security boundaries
- Promote ethical technology practices

---

## Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment or discrimination
- Offensive or abusive language
- Intentional disruption
- Exposure of sensitive information
- Misuse of infrastructure-related content
- Malicious technical activity

---

## Technical Collaboration Standards

Participants should prioritize:

- Technical clarity
- Operational accuracy
- Security awareness
- Documentation quality
- Responsible cloud practices

---

## Enforcement

Repository maintainers reserve the right to moderate contributions, discussions and repository interactions to preserve a professional and respectful environment.

---

## Scope

This Code of Conduct applies to:

- Repository discussions
- Pull requests
- Issues
- Technical collaboration
- Documentation contributions

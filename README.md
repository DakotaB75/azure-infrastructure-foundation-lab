  Executive Overview

This repository contains a structured Microsoft Azure infrastructure environment developed as part of advanced practical training focused on enterprise cloud administration, Azure operational governance and cloud security fundamentals.

The environment was designed following operational methodologies commonly applied within Microsoft Azure consulting ecosystems, emphasizing:

- infrastructure organization,
- governance alignment,
- operational visibility,
- cloud security awareness,
- cost-aware administration,
- and disciplined resource lifecycle management.

Rather than focusing exclusively on theoretical Azure concepts, this project prioritizes hands-on operational execution inside Microsoft Azure Portal using enterprise-oriented deployment practices and documentation standards.

# Governance & Operations

Enterprise cloud environments require significantly more than simply provisioning resources.

For this reason, the environment was designed following operational governance methodologies commonly applied in Microsoft Azure administration and cloud consulting environments.

## Governance Principles Applied

- Dedicated Resource Group segmentation
- Structured naming conventions
- Minimal infrastructure exposure
- Low-cost infrastructure selection
- Controlled deployment scope
- Cost-aware operational management
- Administrative visibility practices
- Operational cleanup discipline
- Temporary resource lifecycle management

# Deployment Workflow

## 1. Azure Resource Group Deployment

Logical infrastructure segmentation designed for centralized operational management and governance visibility.

---

## 2. Azure Networking Configuration

Deployment of Azure Virtual Networks and subnet structures for internal communication boundaries and infrastructure organization.

---

## 3. Network Security Management

Implementation of Azure Network Security Group (NSG) administrative filtering rules and controlled access boundaries.

---

## 4. Infrastructure Provisioning

Deployment of low-cost Linux infrastructure workloads using Microsoft Azure Virtual Machines following cost-aware operational practices.

---

## 5. Operational Validation

Connectivity verification, infrastructure visibility testing and resource validation procedures executed directly within Azure Portal.

---

## 6. Lifecycle Management

Controlled resource cleanup procedures implemented to maintain governance standards and prevent unnecessary Azure subscription consumption.
